This is a demo site created with pdx.py, a geodjango project for building geographic web applications for Portland using the python language. It comes preloaded with data released by local government agencies in association with the CivicApps challenge.

pdx.py is an implementation of the Portland Application Development Framework, which was selected for an award as part of the Idea phase of the 2010 CivicApps Challenge.

pdx.py consists of:
  • Django, a mature and powerful open source web development framework for python with GIS extensions
  • Object-relational models of CivicApps datasets, which can be spatially queried
  • Methods for loading CivicApps datasets from ESRI shapefiles into PostGIS, a powerful relational geodatabase based on PostgreSQL
  • An administrative interface for editing the geographic features and attributes of the models
  • Web Mapping templates using the Google Maps API, Open Layers and Open Street Maps
To try out the Admin control panel, use username: demo, password: demo to authenticate.

This project has been submitted as an App for the 2010 CivicApps Challenge.
Feel free to use pdx.py for as a starting point for creating your own Portland specific web applications. The code is available as an open source project on GitHub and contributors are welcome.
